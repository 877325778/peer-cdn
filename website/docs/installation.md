*_PeerCDN_* is a lightweight library providing peer to peer CDN functionality.

## Installation

<!--DOCUSAURUS_CODE_TABS-->
<!--npm-->
```yarn
npm install peer-cdn
```
<!--yarn-->
```bash
yarn add peer-cdn
```
<!--END_DOCUSAURUS_CODE_TABS-->
